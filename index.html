BuildVITs

A Community-Driven Project-based Learning Initiative

It would be a community-driven project-based learning initiative where we would be able to build, teach, support and collaborate amongst ourselves to solve problems and create things we love!! We would like to nurture the projects made by students. We will be hosting webinars activities and would like to actively collaborate with other clubs so that one can learn from there and apply it here. This club will extend its hands from art, DIY, software projects to drones. Anything techy, and requested by the community Here the creativity in different fields would be encouraged rather than a specific area. It's a community over competition.

Mobile Application Development

We focus that the project based learning includes the making of different Mobile applications through Flutter, React-Native, Kotlin Etc.

Web Application Development

The projects will also include web applications development through different platforms for web developers through React, Django, MERN, Etc.

Other Development

There will be more things different types of projects for different developers who are more interested in other things like IOT, Drones, Electronics, Etc.
